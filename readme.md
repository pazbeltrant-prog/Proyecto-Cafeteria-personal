*Descripción de la tienda de manga y libros*
- La aplicación está construida con Vue 3, Vite y Bootstrap 5, y presenta un diseño de alta calidad con suaves degradados y una tipografía moderna.

*Funcionalidades implementadas*
1. Catálogo de productos
- Categorías: incluye manga, libros y accesorios de lectura.
- Aspecto visual: cada producto cuenta con una imagen de alta calidad, nombre, precio y etiqueta de categoría.
- Interactividad: efectos al pasar el cursor y función «Añadir al carrito».
2. Sistema de carrito de la compra
- Carrito en la barra lateral: una elegante barra lateral para gestionar los productos seleccionados.
- Funcionalidad:
- Añadir varios artículos.
- Aumentar o reducir cantidades.
- Eliminar artículos.
- Cálculo automático del total.
- Compra simulada: un botón de pago con confirmación de éxito.
3. Servicio de cafetería
- Una sección dedicada que muestra el menú de la cafetería con un moderno tema oscuro y una imagen de fondo.
4. Reservas de salas
- Un formulario de reserva para clubes de lectura y talleres.
- Incluye validación y respuesta simulada al envío.
5. Diseño adaptativo y pie de página
- Diseño totalmente adaptativo para móviles y ordenadores de sobremesa.
- Pie de página completo con datos de contacto, enlaces a redes sociales e integración con - - Google Maps (simulada).
